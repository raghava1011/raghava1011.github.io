Large Language Models are Efficient Learners of Noise-Robust Speech Recognition

Team: DO_DIN_MAI_RESEARCH_DOUBLE

Date: 20th April, 2024

In recent years, there have been several advancements in refining Automatic Speech Recognition (ASR) techniques. In this blog, we will discuss an approach that utilizes the powerful capabilities of large language models (LLMs) to improve ASR performance in noisy environments significantly. Generative Error Correction (GER) is the latest benchmark that helps ASR systems predict true transcriptions from the input speech. But GER doesn’t give results at the human parity in noisy conditions. This blog explains the work of Hu et al., (2024) which is an enhancement to the GER framework called as the RobustGER that can handle speech in noisy environments.

What is Generative Error Correction (GER)

The traditional ASR systems generate the transcriptions auto-regressively from the input speech by selecting the most probable token at every time step. Instead of doing that, the GER framework (Chen et al., 2024) utilizes the HyPoradise (HP) dataset, which has the N-best hypothesis and ground truth transcription pairs to directly learn a mapping from speech to true transcription, as shown in the figure below. GER learns Hypotheses to transcription (H2T) mapping 𝓜H2T𝓜_{H2T}𝓜H2T​﻿ to predict the transcription YYY﻿

𝓜H2T𝓜_{H2T} 𝓜H2T​﻿ mapping can be learned from fine-tuning the LLMs using cross-entropy loss LH2T\mathcal{L}_{H2T}LH2T​﻿ with the help of ground truth transcription Y∗Y^*Y∗﻿.

N-best hypotheses are generated using state-of-the-art models like WavLM, Whisper etc, and using beam search decoding on them. The mapping is learned using LLMs like T5, LLama, etc in the In-Context Learning (ICL) setting or their parameter can be finetuned.

Experiments have been conducted by Hu et al.,(2024) to extend the GER to noisy conditions. They proposed the Robust HyPoradise (Robust HP) dataset with 113k pairs of hypothesis-transcription pairs from various ASR corpus in noisy scenarios. They observed the performance gain of GER in noisy conditions is limited which indicates GER is still prone to source audio noise. To overcome this want to make the system noise aware.

Robust GER

LLMs can become noise-aware by conditioning their generation on input audio noise information. However, studies by Zhang et al., (2023) suggest that introducing heterogeneous modalities could cause performance degradation and instability due to the cross-modality gap.

Fig-2 shows transcriptions generated by the GER and GER+Audio Denoising frameworks on a noisy speech signal. We can see that GER+Audio denoising generates a better transcription than the GER framework. But still, the GER+Audio Denoising framework transcription is inaccurate. We can attribute this to the cross-modality gap between the text and the audio.

In response, Hu et al., (2024) introduced a noise embedding in language (text) space to represent the noise in the input audio signal. This noise embedding is based on the diversity of the N-best Hypotheses obtained beforehand. Additionally, they enhance the noise representational capabilities of the noise embedding by distilling knowledge from the input audio. The noise embedding and N-best hypotheses are then sent to an LLM. Then LLM learns to denoise the noise embedding and generate accurate transcription using N-best hypotheses. More on Language Space Noise Embedding

As mentioned before noise embedding in language space is constructed using N-best hypotheses. The main intuition behind it is that the more noisy the input audio is more diverse the N-best hypotheses are. Noise embedding is extracted at two levels: Utterance level and token level.

Utterance level:\textit{Utterance level:}Utterance level:﻿ Analyzes diversity between N-best hypotheses at the semantic level. First sentence embeddings are obtained for the hypotheses set YN\mathcal{Y_N}YN​﻿ using the Sbert encoder Esbert\mathcal{E}_{sbert}Esbert​﻿ and then the difference is calculated between every pair of hypotheses YjY_jYj​﻿ and other hypotheses ranked less than it ( Yi;∀i>jY_i; \forall i>j Yi​;∀i>j﻿ ). Because hypotheses with less index have better rank and are thus close to true transcription and less noisy. All the resultant embeddings after the difference are concatenated and made into one final embedding ELNuttE_{LN}^{utt}ELNutt​﻿ to represent utterance level noise.

Token level:\textit{Token level}:Token level:﻿ Token level noise is measured by the WER metric. All the hypotheses are force-aligned to the same length and the edit distance is measured as shown in the below figure.

The edit distance is measured between each pair of hypotheses and they are concatenated together to form token-level noise embedding. The token level noise embedding is represented using ELNtokE_{LN}^{tok}ELNtok​﻿ and corresponds directly to the gap in the final WER performance of the N-best hypotheses.

The final language noise embedding is the result of concatenation between utterance level noise embedding and token level noise embedding.

Audio Noise Distillation

The audio Noise distillation process helps to increase the noise representational capabilities of language space noise embedding. This distillation is done via Mutual Information estimation. Mutual Information is a measure of dependence between random variables based on Shannon entropy. The calculation of MI is intractable. So we use an estimation called MINE (Mutual Information Neural Estimation) (Belghazi et al., 2018). MINE uses a statistical network to which noise embedding, clean speech, and noisy speech are given. MINE maximizes the mutual information between the noisy audio and language space noise embedding.

Conclusion

In conclusion, the RobustGER approach represents a significant advancement in noise-robust automatic speech recognition by leveraging the impressive capabilities of large language models. By cleverly extracting language-space noise embeddings from ASR hypothesis diversity and employing knowledge distillation, RobustGER enables LLMs to effectively "denoise" transcriptions in the language domain. Achieving up to 53.9% word error rate reductions on noisy speech, work by Hu et al., (2024) demonstrates how LLMs can act as efficient learners robust to multi-modal noise and distortions. As LLMs continue advancing, innovative multi-modal approaches like RobustGER will likely drive further breakthroughs by seamlessly bridging representations across modalities. For real-world speech interfaces and beyond, RobustGER brings us closer to handling the rich complexity of noisy environments.

Key Takeaways:

  1. RobustGER extends generative error correction to noisy speech by making LLMs explicitly noise-aware through language-space conditioning.
  1. A language-space noise embedding extracted from N-best hypothesis diversity patterns captures audio noise characteristics.
  1. Knowledge distillation via mutual information estimation enhances the embedding's noise representational capabilities.
